# Лабораторна робота №6 - Взаємодія Frontend-Backend

## Опис
Динамічний компонент Tabs з можливістю створення та перегляду вкладок.

## Технології
- HTML5, CSS3
- Vanilla JavaScript (без фреймворків)
- PHP 8.x
- JSON для зберігання даних

## Посилання на робочу версію
https://catspage.lovestoblog.com/

## Встановлення
1. Завантажте файли на PHP-хостинг
2. Переконайтесь у правах доступу до папки (755)
3. Відкрийте page1.html для створення вкладок
